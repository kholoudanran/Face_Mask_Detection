##📋 Overview
This project implements a real-time face mask detection system using YOLOv11 (You Only Look Once), a state-of-the-art object detection model. The model is trained to detect whether people are wearing masks, not wearing masks, or wearing masks incorrectly.
## Prerequisites
Python 3.8 or higher

CUDA-compatible GPU (optional but recommended)

pip package manager
